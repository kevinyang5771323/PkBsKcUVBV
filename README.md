# 项目概述

本项目是一款基于SpringBoot和Vue的物业管理系统，适用于小区的综合管理。

## 技术栈

- Java
- SpringBoot
- MyBatis-Plus
- Redis
- Vue
- IDEA
- JDK1.8
- Maven
- Node.js
- MySQL
